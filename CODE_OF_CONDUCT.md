# Code of Conduct

## Our Standard

We are committed to a respectful, inclusive, and harassment-free community.

Participants are expected to:

- be respectful and constructive
- assume good intent and collaborate professionally
- provide actionable feedback
- avoid personal attacks, discrimination, and harassment

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of conduct.
They may remove or edit comments, issues, or contributions that violate these standards.

## Scope

This code of conduct applies in project spaces, including issues, pull requests, and related discussions.
